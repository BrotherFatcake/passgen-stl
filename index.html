How we do it

Checking Passwords:

We hash the password you enter and send the first five characters of the hash to 'haveibeenpwned.com'. In return they match those five characters and return to us all hash values which started with the first five characters we initially sent. From there a comparison is done on the client side and we inform you if the password entered was present in the haveibeenpwned.com database.


Creating new Passwords:

We give you the choice of a traditional password or a passphrase. Once you determine the type you select some options and we send a request to 'makemeapassword.ligos.net' or 'randomuser.me'. They in return generate and return a password based on your specified parameters. Once the password is recieved by the client we run it through the Check Password process for good measure.
