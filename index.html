How we do it:


Checking Passwords:

We convert the password you enter into a new alphanumeric string and send the first five characters of that string to 'haveibeenpwned.com'. In return, they match those five characters and return to us all string values which started with the first five characters we initially sent. From there a comparison is done in the browser and we inform you if the password entered was present in the 'haveibeenpwned.com' database.


Creating New Passwords:

We give you the choice of a traditional password or a passphrase. Once you determine your preferred type, you select different options and we send a request to 'makemeapassword.ligos.net' or 'randomuser.me'. They, in return, generate and return a password based on your specified parameters. Once the password is received by the browser we run it through the Check Password process for good measure.
